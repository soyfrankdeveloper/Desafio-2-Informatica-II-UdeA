El proyecto consiste en modelar y simular el Mundial de la FIFA 2026 mediante el desarrollo de la plataforma UdeAWorldCup. El sistema no emulará las dinámicas internas del juego, sino que funcionará como un motor estadístico y de gestión de datos estructurado bajo el paradigma de Programación Orientada a Objetos (POO).
